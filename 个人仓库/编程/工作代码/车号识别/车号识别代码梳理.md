## 数据库中的DHG_JK_CPSB表内字段构成
```json
RQSJ：日期时间
JCXH：进车序号
CX：车型
CH：车号
DX：道线
FX：方向
DBWNAME：报文名
BZW ：标志位
```
其中的BZW解释
```json
0:将车辆信息插入数据库时状态
1:从报文中提取出时的状态
2:
3:
```

获取文件名称的方法
```csharp
public string Get_FileName(string sDX, string sNUM)
{
  int num = Convert.ToInt32(sNUM.Trim()) + 1;
  if (num > 0 && num <= 999)
  {
    string str = Convert.ToString(num);
    if (str.Length == 1)
      str = "00" + str;
    if (str.Length == 2)
      str = "0" + str;
    return MainForm.G_File_Top.Trim() + sDX.Trim() + str.Trim() + this.Get_FileNameLastDay();
  }
  MainForm.G_Error_Msg_Show += "D报文编号超出规定范围！";
  return "0";
}

对于A道线来说：C000 1 004 .0924
C000:通用开始头
1：道线序号
004：道线最后数字(统一为3位数，且会根据当天的送车批次进行更新)
.0924:当前日期

```
